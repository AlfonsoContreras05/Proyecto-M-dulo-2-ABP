# Alke Wallet

Proyecto desarrollado para la evaluación del **Módulo 2: Fundamentos del desarrollo Frontend**.

## Descripción

**Alke Wallet** es una aplicación web tipo billetera digital que permite simular operaciones financieras básicas desde una interfaz simple, responsiva y fácil de usar.

El proyecto permite iniciar sesión, visualizar un saldo disponible, realizar depósitos, enviar dinero de forma simulada y revisar un historial de movimientos.

## Objetivo del proyecto

El objetivo principal es construir una vista funcional de una wallet digital utilizando tecnologías frontend, aplicando estructura HTML, estilos CSS, componentes de Bootstrap, lógica con JavaScript e interacciones con jQuery.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap
* jQuery
* Git
* GitHub

## Funcionalidades principales

* Inicio de sesión simulado.
* Visualización de saldo disponible.
* Depósito de dinero.
* Envío de dinero a contactos.
* Registro de movimientos.
* Historial de transacciones.
* Diseño responsive adaptable a distintos dispositivos.
* Interacciones dinámicas con JavaScript y jQuery.

## Pantallas del proyecto

El proyecto cuenta con las siguientes vistas:

* `index.html`: página inicial/redirección.
* `login.html`: pantalla de inicio de sesión.
* `menu.html`: menú principal de la wallet.
* `deposit.html`: pantalla para realizar depósitos.
* `sendmoney.html`: pantalla para enviar dinero.
* `transactions.html`: pantalla de historial de movimientos.

## Decisiones tomadas

Para este proyecto se decidió trabajar con una estructura separada por pantallas, lo que permite organizar mejor cada funcionalidad de la wallet.

Se utilizó Bootstrap para acelerar la creación de una interfaz responsive y componentes visuales reutilizables. Además, se incorporó CSS personalizado para mejorar la identidad visual del proyecto.

JavaScript se utilizó para manejar la lógica principal de la aplicación, como validación de login, actualización de saldo, depósitos, transferencias y registro de movimientos.

jQuery fue integrado para facilitar la manipulación del DOM, mostrar mensajes dinámicos y mejorar algunas interacciones visuales.
